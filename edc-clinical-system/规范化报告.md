## 使用指南

### 使用 CSS 变量

```tsx
// 内联样式
<div style={{ color: 'var(--color-primary-500)' }}>

// CSS 模块
.myClass {
  color: var(--color-primary-500);
  padding: var(--space-md);
  border-radius: var(--radius-md);
}
```

### 可访问性最佳实践

```tsx
// 可点击元素
<div
  role="button"
  tabIndex={0}
  aria-label="描述性标签"
  onClick={handleClick}
  onKeyDown={(e) => {
    if (e.key === 'Enter' || e.key === ' ') {
      e.preventDefault()
      handleClick()
    }
  }}
>

// 图标按钮
<Button
  icon={<EyeOutlined />}
  aria-label="查看详情"
  style={{ minWidth: 44, minHeight: 44 }}
/>
```
